## Editor

**VS Code** with these essential extensions:

- Vue - Official
- Tailwind CSS IntelliSense
- ES7+ React/Redux/React-Native snippets
- GitLens
- Prettier

### Theme

I use **One Dark Pro** for a comfortable dark coding experience.

## Terminal

**Windows Terminal** with PowerShell 7 and Oh My Posh for the prompt.

## Key Tools

### Version Control

```bash
git status        # Check changes
git add .         # Stage all
git commit -m     # Commit with message
```

### Package Manager

**Bun** for fast package management:

```bash
bun install       # Install dependencies
bun run dev       # Start dev server
bun build         # Build for production
```

### Browser

**Firefox Developer Edition** for testing and debugging.

## Design Philosophy

> "The best tool is the one you're comfortable with."

I prioritize:

1. **Speed** - Fast startup, quick actions
2. **Keyboard-first** - Minimize mouse usage
3. **Minimalism** - Only what I need
4. **Consistency** - Same shortcuts across apps

## Daily Workflow

1. Open terminal in project folder
2. `bun run dev` to start development
3. Code with music playing
4. `git add` and `commit` throughout the day
5. `bun build` before pushing
